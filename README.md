# 🎓 Student Performance Prediction

A Machine Learning project that analyzes academic factors and predicts whether a student is likely to pass or fail using Logistic Regression.

## 🚀 Project Overview

This project leverages Machine Learning techniques to evaluate student performance based on key academic indicators such as:

* Study Hours
* Attendance Percentage
* Previous Academic Scores
* Assignment Completion Rate

The model is trained to identify patterns in student behavior and generate predictions with confidence scores, helping demonstrate how data-driven insights can support educational decision-making.
